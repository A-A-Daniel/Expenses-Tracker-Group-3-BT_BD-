Expense Tracker API
A lightweight Express.js API designed to manage personal finances through a complete set of CRUD operations for tracking expense amounts, dates, and categories. This project serves as a functional backend demonstration of RESTful architecture using in-memory data storage.

How to Run and Test
Setup: Run pnpm install in your terminal to install dependencies, then start the server using node index.js.

Testing: Open Postman and point your requests to http://localhost:3000/expenses to test the POST, GET, PATCH, and DELETE endpoints.
